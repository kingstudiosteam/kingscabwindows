THE KING'S CAB - WINDOWS INSTALLATION PACKAGE
==============================================

QUICK INSTALLATION:
1. Right-click on "INSTALL.bat"
2. Select "Run as Administrator"
3. Follow the prompts
4. Restart your DAW

ALTERNATIVE INSTALLATION:
1. Open PowerShell as Administrator
2. Navigate to this folder
3. Run: .\TheKingsCab-Windows-Installer.ps1

WHAT GETS INSTALLED:
• VST3 Plugin → C:\Program Files\Common Files\VST3\
• AAX Plugin → C:\Program Files\Common Files\Avid\Audio\Plug-Ins\
• Registry entries for proper Windows integration
• Uninstaller entries in Control Panel

COMPATIBLE DAWs:
✓ Pro Tools (AAX)
✓ Reaper (VST3)
✓ Cubase (VST3)
✓ Studio One (VST3)
✓ FL Studio (VST3)
✓ Ableton Live (VST3)
✓ And all other VST3 compatible DAWs

SYSTEM REQUIREMENTS:
• Windows 10 (64-bit) or later
• 4GB RAM minimum
• 50MB free disk space
• Administrator privileges for installation

UNINSTALLATION:
To uninstall, run:
powershell.exe -ExecutionPolicy Bypass -File "TheKingsCab-Windows-Installer.ps1" -Uninstall

Or use Windows "Add or Remove Programs" in Settings.
